Introduction

Travel and tour Api

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer {YOUR_AUTH_KEY}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting your dashboard and clicking Generate API token.

Admin endpoint

POST Travel

requires authentication

create a new Travel record.

Example request:
curl --request POST \
    "http://localhost/api/v1/admin/travels" \
    --header "Authorization: Bearer {YOUR_AUTH_KEY}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"is_public\": true,
    \"name\": \"test name\",
    \"description\": \"test description\",
    \"number_of_days\": 5
}"

const url = new URL(
    "http://localhost/api/v1/admin/travels"
);

const headers = {
    "Authorization": "Bearer {YOUR_AUTH_KEY}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "is_public": true,
    "name": "test name",
    "description": "test description",
    "number_of_days": 5
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):

GET Tour

Get specific Tour record.

Example request:
curl --request GET \
    --get "http://localhost/api/v1/travels/996b7ad5-b57a-498c-8568-0b388b42c1e6/tours?priceFrom=2000&priceTo=900&dateFrom=2023-06-18&dateTo=2023-06-18&sortBy=officiis&sortOrder=eius" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"priceFrom\": 8.389872996,
    \"priceTo\": 79540.1043,
    \"dateFrom\": \"2023-06-19T11:11:57\",
    \"dateTo\": \"2023-06-19T11:11:57\",
    \"sortBy\": \"price\",
    \"sortOrder\": \"desc\"
}"

const url = new URL(
    "http://localhost/api/v1/travels/996b7ad5-b57a-498c-8568-0b388b42c1e6/tours"
);

const params = {
    "priceFrom": "2000",
    "priceTo": "900",
    "dateFrom": "2023-06-18",
    "dateTo": "2023-06-18",
    "sortBy": "officiis",
    "sortOrder": "eius",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "priceFrom": 8.389872996,
    "priceTo": 79540.1043,
    "dateFrom": "2023-06-19T11:11:57",
    "dateTo": "2023-06-19T11:11:57",
    "sortBy": "price",
    "sortOrder": "desc"
};

fetch(url, {
    method: "GET",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
